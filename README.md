# AI / 数据作品集诊所

面向应届、转行和早期职业阶段求职者的小范围材料优化服务。目标不是把材料写得更“像 AI”，而是让招聘方在 30 秒内看懂：你解决了什么问题、贡献了什么、证据在哪里，以及哪些内容是真实实现、哪些只是演示。

## 主推服务

1. **AI / 数据作品集 15 分钟体检：69 元**
   - 检查 GitHub README、项目首页或作品集页面中的一个
   - 给出第一印象评分、3 个最大问题、按优先级排序的修改清单
   - 附送一段可直接替换的项目介绍
   - 24 小时内交付 PDF 或 Markdown

2. **简历 × 目标 JD 精修：129 元**
   - 精修 1 页中英文简历中的一种语言版本
   - 分析目标 JD 关键词与证据缺口
   - 重写项目/经历表达，不虚构经历
   - 交付可编辑版 + 修改说明，24 小时内完成

## 成交规则

- 只承诺材料诊断和改写，不承诺面试或 offer。
- 不代写虚假经历、不伪造数据、不碰客户账号密码。
- 客户先发脱敏材料确认范围；确认后按平台担保或双方认可的方式付款。
- 每单默认含 1 次小范围修改，交付后 48 小时内提出。

## 咨询方式

请使用仓库的 **Issues → New issue → 服务咨询**。只填写目标方向和公开项目链接，不要上传简历、电话、邮箱、住址或其他个人信息。确认服务范围后，再通过双方认可的私密方式传递脱敏材料。

> 这是材料诊断与表达优化服务，不是招聘中介，不承诺面试或 offer。
